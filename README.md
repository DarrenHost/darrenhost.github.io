## 🚀 本地预览

```bash

# 进入目录
cd darrenhost.github.io

# 使用 Python 启动本地服务器

# 访问 http://localhost:8080
```

## 🔧 部署方式

本站使用 **GitHub Pages** 自动部署：

1. 推送更改到 `main` 分支
2. GitHub 自动构建并部署
3. 访问 https://DarrenHost.github.io

### 自定义域名（可选）

如需使用自定义域名：

1. 在仓库根目录创建 `CNAME` 文件
2. 写入域名（如：`blog.darrenhost.com`）
3. 在 DNS 提供商处配置 CNAME 记录
